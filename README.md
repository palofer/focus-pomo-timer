# ⏳ FocusPomo

A minimalist, professional Pomodoro tracker designed to enhance deep work and productivity. 

## 🚀 Key Features
* **Customizable Intervals:** Set your own focus and break times.
* **Audio Alerts:** Get notified when a session ends.
* **Persistent Sessions:** (Opcional si lo tienes) Your progress stays even after refreshing.
* **Responsive Design:** Optimized for desktop and mobile.

## 🛠️ Built With
* **TypeScript** - For type-safe, scalable code.
* **React** - Component-based UI library.
* **CSS Modules / Tailwind** - (Pon el que uses) For sleek, modern styling.

## 📦 Installation & Setup
1. Clone the repo: `git clone ...`
2. Install dependencies: `npm install`
3. Run the app: `npm start`
